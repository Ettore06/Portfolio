Smarketech

Gestor de mercado

O Smarketech é um sistema de gestão desenvolvido com o objetivo de centralizar e organizar informações de forma estruturada, garantindo consistência dos dados e facilitando a tomada de decisão. O projeto foi pensado desde o início com foco em back-end, banco de dados e integração funcional, priorizando uma arquitetura clara e escalável.

Tela de login

Tela de recuperação de senha

O fluxo de autenticação seguro permite o acesso seguro às funcionalidades do programa de gestão. A interface de login se comunica com o back-end para validação das credenciais, enquanto o fluxo de recuperação de senha permite a redefinição de acesso de forma controlada.

Tela de gestão de produtos

A aplicação permite o gerenciamento completo do catálogo de produtos, incluindo cadastro, visualização, edição e exclusão. O front-end consome endpoints da API para manter os dados sincronizados, refletindo alterações de forma dinâmica na interface.

A organização em cards facilita a visualização das informações essenciais, como preço, estoque e validade, tornando o uso do sistema mais intuitivo.

Tela de relatórios

O dashboard de relatórios apresenta informações consolidadas do sistema, como total de produtos, valor em estoque, alertas e tendências. Os dados exibidos são processados no back-end e apresentados no front-end por meio de gráficos e indicadores visuais.

Essa camada analítica auxilia na interpretação dos dados e na tomada de decisão, transformando informações brutas em insights visuais.

Tela de histórico de pedidos

A aplicação possui uma área dedicada ao acompanhamento de pedidos, permitindo visualizar histórico, status, valores e itens vendidos. O front-end realiza filtros e consultas que são processadas pelo back-end, retornando dados organizados e atualizados.

Essa funcionalidade centraliza o fluxo operacional do sistema, conectando produtos, clientes e valores de forma estruturada.
