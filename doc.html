Smarketech

A documentação do Smarketech foi desenvolvida com o objetivo de registrar o funcionamento do sistema de forma clara, técnica e acessível, servindo como apoio para o entendimento da aplicação e para a integração entre front-end e back-end.

A API utilizada foi documentada por meio do Swagger, permitindo a visualização dos endpoints, métodos HTTP, parâmetros e respostas de forma organizada. Essa documentação facilita o consumo da API, além de possibilitar testes e validações das rotas durante o desenvolvimento.
